Proyecto de ejemplo: Bitácora (Baruc Publicidad)
Instrucciones rápidas:
1) Instala Expo CLI si no lo tienes: npm install -g expo-cli
2) Copia la carpeta 'bitacora-baruc' a tu máquina de desarrollo.
3) Dentro de la carpeta ejecuta: npm install
4) Inicia el proyecto: npm run start
Dependencias clave ya listadas en package.json.
Nota: las imágenes del logo están en /assets (logo.png, logo@2x.png, logo@3x.png)
